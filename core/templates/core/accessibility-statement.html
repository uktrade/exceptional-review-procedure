Accessibility statement for Give feedback on the UK’s temporary tariff regime rates

This online form is run by the Department for International Trade. We want as many businesses as possible to be able to use this service. For example, that means you should be able to:

  • zoom in up to 300% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
    • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made the website text as simple as possible to understand. AbilityNet has advice on making your device easier to use if you have a disability.

How accessible this website is

We are in the process of arranging a full WCAG 2.1 audit for this online form. We will update this statement once we’ve received the WCAG 2.1 audit report. In the meantime, we have done some basic checks, and addressed the issues that were flagged.

We used components in the GOV.UK Design System to build this form. The Design System accessibility guide explains the current state of accessibility for these components. We are using a modified API from HMRC’s Trade Tariff: look up commodity codes, duty and VAT rates tool to allow users to search for commodity codes. We will test the accessibility of this component but fixing any issues will rely on HMRCs development plan.

What to do if you cannot access parts of these services

If you need information on this website in a different format like accessible PDF, large print, easy read, audio recording or braille:

  • Email ERPFeedback@trade.gov.uk
  • Telephone: +44 (0)20 7215 5000 Monday to Friday, 9am to 5pm

Reporting accessibility problems with this website

We’re always looking to improve the accessibility of this website. If you find any problems not listed on this page or think we’re not meeting accessibility requirements, contact:

  • help@digital.trade.gov.uk

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about these services’ accessibility

The Department for International Trade is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This website is partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard, due to the non-compliances listed below.

Non accessible content

All content for this service is designed by an experienced team of content designers to ensure we meet the highest web content standards.

The content listed below is non-accessible for the following reasons:

Non compliance with the accessibility regulations

  • Some of the forms elements might not be fully compatible with screen readers.
  • Some of the autocomplete form fields might not be fully compatible with screen readers.
  • We use Google reCAPTCHA that may not be fully accessible as it is JavaScript generated.

We plan to fix these problems by September 2020.

There are known issues with the GOV.UK Design System, as explained in the Design System accessibility guide. We will update our form with any accessibility fixes once they are released to the GOV.UK Design System.

How we tested these services

Basic tests were performed on this online form on Thursday 7 November. The form was tested from start to finish with Pa11y. These tests were carried out by the Department for International Trade.

What we’re doing to improve accessibility

We are in the process of arranging a full WCAG 2.1 audit of this online form. We will then update this statement and aim to implement any changes by September 2020.

There are known issues with the GOV.UK Design System, as explained in the Design System accessibility guide. We will update our form with any accessibility fixes once they are released to the GOV.UK Design System.

This statement was prepared on 13 November 2019. It was last updated on 13 November 2019.
